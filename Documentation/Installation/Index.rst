Installation guide
==================

Install the extension using your preferred method.

Head over to your CLI and run the following command

.. code-block:: bash

	composer require maxserv/frontend-request

Not using composer? Download the zip file from the
`TER <https://extensions.typo3.org/extension/frontend_request>`__ and upload it through
the Extension Manager.
