Contribution
============

Anyone is welcome to contribute to Frontend Request.

There are various ways you can contribute:

*  `Raise an issue <https://github.com/MaxServ/frontend-request/issues>`__ on GitHub.
*  Create a pull request with your bug fixes or new features.
*  Improve this documentation via the "Edit on GitHub" workflow.

Please `read the guidelines <https://github.com/MaxServ/frontend-request/blob/main/.github/CONTRIBUTING.md>`__
on how to contribute to this repository and how to report bugs.
